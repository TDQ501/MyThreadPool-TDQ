# MyThreadPool-TDQ
这是一个基于 C++11 标准库实现的 MySQL 数据库连接池。它通过复用一组预先创建的连接，显著降低了频繁建立/断开 TCP 连接的开销，在多线程高并发场景下，能大幅提升数据库操作效率。
核心特性：
线程安全：基于 `mutex` + `condition_variable` 实现生产者-消费者模型。
动态扩容：连接数可在 `initSize` 和 `maxSize` 之间动态调整。
空闲回收：独立线程定期扫描并释放空闲超时的连接。
连接超时：获取连接时支持超时等待，避免死等。
RAII 管理：返回 `shared_ptr<Connection>`，自动归还连接到池中。
